# CLAUDE.md

## 目录职责

- 本目录存放面向 AI 代理的渠道提示和技能资产。
- 它只定义“如何在本仓库/本渠道工作”，不直接承载业务实现。

## 可以在这里放

- 运行时 prompt、正式 `SKILL.md`、技能配套说明。
- 明确面向 AI 代理消费的结构化指导文档。

## 不要在这里放

- 产品源码、构建产物、一次性调试日志。
- 会被直接当成运行时 prompt 的大而全维护文档。

## 修改约束

- 新增技能资产优先目录化：`skills/<name>/prompt.md` 与 `skills/<name>/SKILL.md` 分工明确。
- `prompt.md` 只写当前会话所需的最小事实和边界；`SKILL.md` 才承载说明、维护和发现职责。
- `prompt.md` MUST NOT 包含“何时发卡”“如何组织标题/摘要/结论”“按钮推荐文案”“发送前自检”等会塑形 agent 输出策略的内容。
- 过渡期允许保留历史单文件资产，但新增内容不要继续扩展单文件混合模式。
